# Directives (Layer 1: SOPs)

This directory contains Standard Operating Procedures (SOPs) written in Markdown. These documents define the goals, inputs, tools to use, and expected outputs for specific tasks.

## Structure of a Directive

1.  **Goal**: What is the objective of this SOP?
2.  **Inputs**: What data or parameters are required?
3.  **Tools/Scripts**: Which deterministic scripts in `execution/` should be called?
4.  **Process**: Step-by-step instructions for the Orchestrator (Decision Maker).
5.  **Outputs**: What should be the final result?
6.  **Edge Cases**: How to handle common errors or exceptions.
